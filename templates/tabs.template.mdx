## Overview

A set of layered sections of content—known as tab panels—that are displayed one at a time.

Tabs organize related content into separate views, allowing users to switch between them without leaving the page.

<br/>
Features a smooth animated indicator that highlights the active tab.

Built on top of Base UI Tabs component.
